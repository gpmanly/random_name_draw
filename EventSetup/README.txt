PTC Christmas Party Raffle Setup
============================================================

Preparation:
------------------------------------------------------------
1. Extract the contents of the package to the same directory
as index.html

2. Ensure that the file path does not contain any spaces.
Move the directory as close to root drive
------------------------------------------------------------

Usage:
------------------------------------------------------------
1. Run the raffleSetup executable.

2. Restore the kiosk window before pressing Enter to continue.

3. The shell window will terminate immediately.
------------------------------------------------------------
